# Faker::Company

```ruby
Faker::Company.name #=> "Faure EI", "Dumas et Gauthier"

Faker::Company.suffix #=> "SA", "SAS"

Faker::Company.industry #=> "Sécurité publique", "Textiles"

Faker::Company.buzzword #=> "OpenSource", "Personnalisable"

# Get a random company logo url in PNG format.
Faker::Company.logo #=> "https://pigment.github.com/fake-logos/logos/medium/color/5.png"

Faker::Company.type #=> "Entreprise privée", "Agence Gouvernementale"

# Get a random profession
Faker::Company.profession #=> "Dentiste", "Développeur Web"

# Get a random French siren number
Faker::Company.french_siren_number #=> "819489626"

# Get a random French siret number
Faker::Company.french_siret_number #=> "81948962600013"
```
